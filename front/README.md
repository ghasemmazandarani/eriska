# Eriska Frontend

This directory contains the frontend code, built with **Next.js** and **React**.

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation & Usage

1. Install dependencies:
```bash
npm install
# or
yarn install
```

2. Run the development server:
```bash
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:3000`.

## Technologies

- Next.js 16
- React 19
- TailwindCSS v4
- Zustand (State Management)
- Recharts (Data Visualization)
