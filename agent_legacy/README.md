# Eriska Security Agent

This directory contains the security agent responsible for scanning the network and identifying vulnerabilities.

## Prerequisites

- Python 3.8+
- Administrator/Root access (for low-level network scans)

## Installation & Usage

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the agent:
```bash
python main.py --mode active
```

For a full list of commands:
```bash
python main.py --help
```
